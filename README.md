# Practical-Data-Science-on-the-AWS-Specialization

This repository contains the notebooks and learning materials completed as part of the **Practical Data Science on the AWS Cloud Specialization**, offered by **DeepLearning.AI** and **AWS** through **Coursera**.

The specialization focuses on applying practical data science and machine learning workflows in the AWS cloud, mainly using **Amazon SageMaker**. The courses cover the full machine learning lifecycle, from data ingestion and exploratory analysis to model training, deployment, monitoring, optimization, and human-in-the-loop pipelines.

The main goal of this repository is to organize the hands-on notebooks developed throughout the specialization and document the key topics covered in each course.

## Specialization Overview

This specialization is designed around practical machine learning workflows in cloud environments. It covers how to prepare and analyze large datasets, train and evaluate machine learning models, use automated machine learning tools, manage features, build pipelines, deploy models, monitor performance, and improve model quality using human feedback.

The labs include applied examples using natural language processing models and AWS services such as Amazon SageMaker, SageMaker Autopilot, Feature Store, Pipelines, Model Monitor, Ground Truth, and Augmented AI.

## Courses
**Course 1**: Analyze Datasets and Train ML Models using AutoML

This course introduces the first stages of the machine learning workflow on AWS. It focuses on data preparation, exploratory data analysis, bias detection, feature engineering, and model training using pre-built algorithms and AutoML tools in Amazon SageMaker.

The course provides a practical foundation for working with datasets in the cloud and preparing them for scalable machine learning workflows.

**Course 2**: Build, Train, and Deploy ML Pipelines using BERT

This course focuses on building more complete machine learning workflows using Amazon SageMaker. It covers feature management, model training, model debugging, profiling, evaluation, artifact tracking, and deployment.

A key part of the course is the use of NLP models, including BERT-based workflows, to understand how modern machine learning models can be trained, evaluated, and operationalized in the cloud.

**Course 3**: Optimize ML Models and Deploy Human-in-the-Loop Pipelines

This course covers techniques for improving model performance and reducing operational costs. Topics include hyperparameter tuning, model comparison, A/B testing, deployment monitoring, and automatic scaling.

The course also introduces human-in-the-loop machine learning workflows using Amazon SageMaker Ground Truth and Amazon Augmented AI, showing how human feedback can be used to improve model predictions and generate better training data.
